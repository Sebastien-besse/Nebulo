# 🌌 Nébulo
Nébulo est une application iOS permettant de suivre ses dividendes d’actions tout en explorant un univers spatial gamifié.
Chaque dividende reçu génère de l’énergie cosmique, permettant de progresser dans une galaxie composée de planètes à débloquer.
L’objectif de l’application est de rendre le suivi des investissements simple, visuel et motivant, en combinant finance personnelle et gamification.
